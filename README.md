# Nest Crud

**| Nest JS**

**| Prisma**

**| Jest**

---

## Setup

### clone the repository

```bash
git clone https://github.com/daviramosds/nest-crud
```

### go to the project folder

```bash
cd nest-crud
```

### create a ".env" and a ".env.test" based in the ".env.example"

```bash
touch .env && touch .env.test
```

### install packages

```bash
yarn
```

### create test db

```bash
yarn prisma:test:migrate
```

### run tests

```bash
yarn test
```

### run project

```bash
yarn start:dev
```

### Done!!!
